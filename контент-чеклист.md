## Тексты (этап 5)

### Hero
- [ ] Имя, слоган (RU / EN)

## Изображения

| Файл | Размер | Формат | Назначение |
|------|--------|--------|------------|
| `avatar.png` | ≥400×400 | PNG | Hero |
| `project.png` | ~1200×675 | PNG | **Один** проект |
| `og.png` | 1200×630 | PNG | Шаринг ссылки в соцсетях |

Положите PNG в `assets/images/` — при сборке Pillow сожмёт копии в `dist/assets/images/` (исходники не меняются).

По имени файла подбирается максимальный размер:
- `avatar*` → 400×400
- `og*` → 1200×630
- `project*` → 1200×675

**Сейчас в репозитории:** SVG-плейсхолдеры — заменить `src` в шаблонах на свои PNG.

## Команды

```bash
pip install -r requirements.txt
python build.py
python -m http.server --directory dist
```

RU: http://localhost:8000/ · EN: http://localhost:8000/en/

## Перед первым деплоем

1. Заполнить тексты и URL в `templates/index.ru.html` / `index.en.html`
2. Положить `avatar.png`, `project.png`, `og.png` в `assets/images/` и обновить `src` в шаблонах
3. Заменить `your_username` в контактах и README
4. Push на GitHub → **Settings → Pages → Source: GitHub Actions**
5. После деплоя проверить HTTPS и превью ссылки в Telegram/VK
